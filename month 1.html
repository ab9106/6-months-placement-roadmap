Month 1: Foundation Building Preparation

Coding Basics - Data Structures and Algorithms (DSA)

  • LeetCode (Platform for coding practice)
    Link: https://leetcode.com/
    Recommended for: Arrays, Strings, Linked Lists, Stacks, Queues, Recursion, Backtracking.
  • HackerRank (Platform for coding practice with tutorials)
    Link: https://www.hackerrank.com/domains/tutorials/10-days-of-javascript
    Recommended for: Array and String problems, plus specific DSA sections.
  • CodeChef (Competitive programming platform)
    Link: https://www.codechef.com/
    Recommended for: A mix of easy, medium, and hard DSA problems, contests.
  • GeeksforGeeks - Data Structures and Algorithms (Tutorial and problem-solving guide)
    Link: https://www.geeksforgeeks.org/data-structures/
    Recommended for: In-depth tutorials and examples for arrays, strings, linked lists, and more.

Core CS Fundamentals

Operating Systems (OS)

  • Operating Systems: Three Easy Pieces (Book)
    Link: https://pages.cs.wisc.edu/~remzi/OSTEP/
    Recommended for: In-depth understanding of OS concepts like processes, threads, and synchronization.
  • GeeksforGeeks - Operating Systems (Tutorials on various OS topics)
    Link: https://www.geeksforgeeks.org/operating-systems/
    Recommended for: OS topics including memory management and process scheduling.
  • YouTube: Computerphile - Operating System Videos (Video tutorials on OS concepts)
    Link: https://www.youtube.com/channel/UCYO_jab_esuFRV4b17AJtAw
    Recommended for: Simplified explanations of OS topics.

DBMS (Database Management Systems)

  • SQLZoo (Interactive SQL tutorials)
    Link: https://sqlzoo.net/
    Recommended for: Hands-on SQL practice.
  • GeeksforGeeks - DBMS (Tutorials on SQL, Normalization, ACID, and more)
    Link: https://www.geeksforgeeks.org/database-management-system-database-models/
    Recommended for: Detailed DBMS concepts.
  • W3Schools - SQL Tutorial (Beginner-friendly SQL tutorials)
    Link: https://www.w3schools.com/sql/
    Recommended for: SQL basics, queries, joins, and transactions.

Object-Oriented Programming (OOP)

  • Java Programming: Solving Problems with Software (Free course on Coursera by UC San Diego)
    Link: https://www.coursera.org/learn/java-programming
    Recommended for: OOP concepts with Java.
  • GeeksforGeeks - OOP Concepts (Tutorials on OOP concepts)
    Link: https://www.geeksforgeeks.org/object-oriented-programming-oops-concept-in-java/
    Recommended for: Detailed explanation of inheritance, polymorphism, encapsulation, and abstraction.
  • YouTube: Programming with Mosh (OOP Concepts) (Video tutorial series)
    Link: https://www.youtube.com/watch?v=pTB0EiLXUC8
    Recommended for: Easy-to-understand OOP concepts explained in a beginner-friendly manner.

Soft Skills

  • Coursera: Introduction to Public Speaking (Online course)
    Link: https://www.coursera.org/learn/public-speaking
    Recommended for: Confidence building and communication tips.
  • YouTube: Communication Skills Training (Soft skills and body language tutorials)
    Link: https://www.youtube.com/results?search_query=communication+skills+training
    Recommended for: Improving communication and body language.
  • LinkedIn Learning: Mastering Your Elevator Pitch (Course on introducing yourself confidently)
    Link: https://www.linkedin.com/learning/mastering-your-elevator-pitch
    Recommended for: Learning how to present yourself effectively in interviews.
