Revise Core Topics

Data Structures & Algorithms (DSA)

  • GeeksforGeeks - Data Structures & Algorithms
    Link: https://www.geeksforgeeks.org/data-structures/
    Recommended for: A comprehensive collection of resources for revising and practicing core topics in DSA. Use it for revising algorithms, data structures, and their applications.

Operating Systems (OS)

  • Operating Systems: Three Easy Pieces (Book)
    Link: https://pages.cs.wisc.edu/~remzi/OSTEP/
    Recommended for: An excellent book for revising OS concepts like processes, threads, synchronization, memory management, and file systems.

Database Management Systems (DBMS)

  • Database Management Systems by Korth (Book)
    Link: https://www.amazon.com/Database-Management-Systems-6th/dp/0073523321
    Recommended for: A core book for revising DBMS topics like normalization, SQL queries, transactions, and indexing.

Computer Networks (CN)

  • Computer Networking: A Top-Down Approach (Book)
    Link: https://www.amazon.com/Computer-Networking-Top-Down-Approach/dp/0132856204
    Recommended for: A thorough resource for revising networking concepts, including the OSI model, TCP/IP, HTTP, DNS, and more.

Object-Oriented Programming (OOP)

  • GeeksforGeeks - Object-Oriented Programming in C++
    Link: https://www.geeksforgeeks.org/object-oriented-programming-in-cpp/
    Recommended for: In-depth revision of OOP concepts like inheritance, polymorphism, abstraction, and encapsulation. Use C++ (or your preferred language) for practice.
